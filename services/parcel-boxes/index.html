Common questions

A few quick answers about parcel boxes.

Can you fit a parcel box if I already have one?

Yes. If you already know the unit you want, we can usually install it cleanly.

Do parcel boxes only make sense for big houses?

No. They are often most useful on compact frontages where deliveries need a clear drop point.

What do they pair well with?

Gates, boundary tidy-ups and front path or paving work.
