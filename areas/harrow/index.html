The local patch

Working in Harrow.

Harrow gardens do a lot of work — family lawns, trampolines, sheds, side returns used for bikes and bins. The jobs we see most here are front boundaries that need to look sharp from the street, and back fences that need to survive kids, foxes and winter wind.

Matching what is already there matters in Harrow's older streets, so we will match an existing fence style or gate detail where we can, and tell you straight when replacing the run outright is the better-value option.

FAQ

Harrow questions.

Which parts of Harrow do you cover?

HA1, HA2 and HA3 — Harrow on the Hill, North and West Harrow, Harrow Weald, Kenton and Rayners Lane are all standard patch.

Can you match my existing fence style?

Usually, yes. Bring a photo to the site visit — if a like-for-like match is not sensible, we will say so and quote the alternative.

Do you fit gates to match new fencing?

Yes. A side or garden gate can be built into the same run and the same quote, hung so it actually shuts first time.
